*TCP (transmission control protocol)* - 
тіпа відкритий телефонний дзвінок між клієнтом і сервером (наприклад, пайтон код і mysql server) для безпечної (error-checked) передачі даних. поки дзвінок не завершений - можна ділитись інфою (передавати дані)


*cursor.fetchone()*
як 
    int arr[] = {10, 20, 30};
    int* ptr = arr; (вказівник на перший елемент, далі ptr++)
в c++


*транзакція* - 
це група змін, яка застосується тільки якщо всі вони будуть успішні, а якщо ні, то ролбек


щоб не було затримки (*latency*) відправляють пакет команд (*batch*), а не по одній
